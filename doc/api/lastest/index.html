SilentEye source code documentation

Introduction

SilentEye is a cross-plateform application design for friendly use of steganography, in this case hidding message into pictures or sound. It's provides a pretty interface and an easy integration of new steganography algorithm and cryptography process by using a plug-ins system.

This documentation provides usefull informations on source code hierarchy and help to understand how SilentEye works.
